# Enterprise Booking API

Production-ready Node.js backend using Express.js, MongoDB, Redis, BullMQ, JWT authentication, RBAC, caching, queue workers, and security best practices.

## Features

- JWT Authentication
- Refresh Tokens
- Role-Based Access Control
- Redis Caching
- BullMQ Queue Workers
- Email Queue
- Rate Limiting
- Helmet Security
- Pino Logging
- Morgan Request Logging
- MongoDB + Mongoose
- Centralized Error Handling

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Redis
- BullMQ
- Pino
- JWT
- Nodemailer

## Installation

```bash
npm install
npm run dev
```
